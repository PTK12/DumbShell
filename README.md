# DumbShell
Boring shell in Python that I glued together. Unstable.

## Features
* Auto-indent
* Suggested words
* Custom indent style/size
* Boring prompt

| Feature     | Use                 |
|-------------|---------------------|
| Auto-indent | Hit <Enter>         |
| Un-indent   | Hit <Enter> <Enter> |
| Suggestion  | Hit <Tab> <Enter>   |
