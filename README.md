# DumbShell
Boring shell in Python that I glued together. Unusable.

## Use
Import it then run like so...
```py
import DumbShell
DumbShell.DumbShell().run()
```

## Features
* Auto-indent
* Suggested words
* Custom indent style/size
* Boring prompt

| Feature     | Use                     |
|-------------|-------------------------|
| Auto-indent | Hit \<Enter\>           |
| Un-indent   | Hit \<Enter\> \<Enter\> |
| Suggestion  | Hit \<Tab\> \<Enter\>   |
